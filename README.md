## Running the code

You will need a Python installation.

**Windows:**

```bat
setup_venv.bat
.venv\Scripts\activate
python main.py
```
  

**macOS/Linux:**

```sh
./setup_venv.sh
source .venv/bin/activate
python main.py
```
